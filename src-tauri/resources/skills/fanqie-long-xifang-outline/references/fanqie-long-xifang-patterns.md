# 番茄长篇西方奇幻样本规律

本参考资料来自本地 `番茄小说/对标作品/长篇/西方奇幻/阅读榜` 下的10部样本。每部只收录前10章，属于开头片段，不是完整全文。因此下文的“样本大纲”只总结已展示的前10章；涉及后续发展时，只提供结构性创作推演，不把推演当作原作事实。

## 核心节奏

番茄长篇西方奇幻的前10章，本质上是在连续回答四个问题：主角为什么快死、凭什么活、第一次如何证明、接下来要去哪里。

- 第1章：高压入场。常见形态是必死反派、黑暗异世界、诡异感染、家族驱逐、战争征召、邪教追杀、领地流放或群体穿越。
- 第2-3章：亮出差异。系统、游戏知识、经验条、指南书、羊皮纸、地图视野、前世修炼法、血脉机缘等必须开始实际工作。
- 第4-5章：建立本地规则。通过一次诊疗、调查、狩猎、救人、交易、建设或战斗，让读者理解光暗规则、职业体系、魔法成本、冒险者规矩、族群秩序或领主权力。
- 第6-8章：高风险验证。主角主动承担一次可能死亡的行动，把金手指从“说明书”变成可见结果，同时引来组织、强者、女主、族群或更大危机的注意。
- 第9-10章：第一次跃迁。常见结果是觉醒职业、进入超凡、建立关系、获得身份、掌握关键情报、拥有第一名核心部下，或从新手场景走向学院、教会、冒险者协会、战争、灰雾外圈和领地治理。
- 第10章末：必须开长线钩子。勇者到来、狩魔人登门、感染扩散、世界大战将启、诅咒倒计时、组织追查、幸存者反抗等，都比单纯“继续升级”更有追读力。

## 共性推进机制

- 生存倒计时：必死CG、诅咒寿命、异梦感染、灰雾迷失、腐尸围攻、黑暗吞噬，让主角不能停在原地。
- 规则可验证：外挂不是一句“无敌”，而是通过处决、看病、配药、沐浴神血、猎杀蜘蛛、觉醒神选等场景兑现。
- 信息差换资源：熟悉游戏剧情、知道古代秘闻、能看经验、能读禁忌知识、拥有地图标记，最终都要换成活命资本。
- 身份反差：被嫌弃的反派、落魄邪教贵族、凡人医生、弱小鹰人、懦弱男爵，迅速做出与外界认知相反的结果。
- 本地社会接入：个人求生很快接入学院、教会、冒险者协会、族群王庭、领主制度、超凡序列或城市情报网。
- 世界观随事件展开：先写路灯为什么密、征召为什么残酷、蜘蛛为何异常，再解释背后的神祇、战争、感染和历史。
- 小闭环带大危机：解决一个病人，暴露更大感染；获得超凡材料，背上死亡诅咒；救下女主，触发地下城任务；觉醒部下，必须清理整片领地。

## 常见爽点

- 别人把危险当绝境，主角把危险拆成可执行步骤。
- 外界把主角当废物、疯子、邪教徒、胆小鬼，结果主角最先看懂规则。
- 稀缺知识迅速变现：医术、药剂、战斗技巧、现代组织知识、游戏攻略、职业构建。
- 成长有可视反馈：系统面板、经验条、职业等级、血脉变化、神眷、地图标记、组织评价。
- 主角不是纯被动受害者，会主动设局、试错、谈判、收人、建设、调查或冒险。
- 超凡力量和现实成本并存：法术有材料成本，晋升要冒险，知识可能污染，力量会引来注视。

## 样本开篇大纲

### 《巫师：让你学知识，你杀穿万界？》

卖点是侯爵独子绑定“击杀生命解锁随机被动”的系统，以贵族资源和超强被动同时推进骑士、巫师两条成长线。

- 第1章：十六岁生日激活无限被动系统，先获得长寿、武器大师、指南针，并把领地死刑犯转化为第一批系统资源。
- 第2章：公开处决重犯，完成生命能量收割，同时建立主角果断、懂规则、能利用身份的形象。
- 第3-4章：抽取植物亲和、武者躯体等被动，骑士修炼速度暴涨，以低境界战斗技巧压制更强护卫，第一次证明外挂。
- 第5-7章：家族、父母、巫师觉醒和资质线进入主轴，主角的贵族身份与巫师道路正式接轨。
- 第8-10章：骑士实力、五等资质、巫师接引与外部竞争连续登场，主角离开领地保护层，进入更大的巫师秩序。

结构要点：随机被动必须迅速形成组合，而不是孤立抽奖；贵族权力负责提供资源，巫师世界负责制造更高层压力。

### 《异世界：我的人生开了挂！》

卖点是人类少年在诡秘异世界经营药剂店，系统把怪物需求转成任务；主角用极端冷静与“恐怖式医疗”反向吓退怪物。

- 第1章：用日记列出生存规则，系统激活，黑暗、魂灯、怪物邻居和血锯老师同时亮相。
- 第2-4章：老师外出，主角独自接诊血娃娃；通过诊断、取异物、缝合和收费完成第一个诊疗闭环。
- 第5-7章：新顾客持续上门，主角的专业、狠劲与误解喜剧叠加，技能、知名度和生存资本同步增长。
- 第8-9章：给猪头魔手术并接生鬼婴，主角以自残式应对和恢复药剂证明极端生存能力，再给鬼婴“整容”压制怨气。
- 第10章：烧死鬼被主角的正常操作吓跑，主角获得名声；强大狩魔人登门，把黑店单元剧接向更大人类/怪物秩序。

结构要点：单元病例每次都要同时推进三件事——技能成长、怪物关系、世界规则；恐怖与喜剧靠认知错位产生。

### 《必死反派的学院地下城生存指南》

卖点是81周目玩家穿成拥有1380种死亡CG的恶役贵族，躲进唯一没有死亡记录的学院地下城，却因剧情偏移被迫救下女主并攻略地下城。

- 第1章：被家族驱逐，展示必死身份、地下城避死逻辑和生存天赋；原作女主意外落水，世界线开始偏移。
- 第2章：主角入水救人、击杀魔物并升级，却被勇者女主认出；系统发布攻略地下城的强制任务。
- 第3-5章：主角用游戏情报编造暗夜神使身份，试图洗白原主恶行；谎言意外被神力检测证实，形成身份大反转。
- 第6-8章：女仆、家族、宝石、地下城异常和暗系亲和等线索并进，主角的生存营地转为剧情人物汇合点。
- 第9-10章：女主们重新评估主角，学院与公主等更高身份角色介入，关系线和地下城攻略线同步展开。

结构要点：熟知剧情不能保证安全，必须不断制造“原作没有这一段”的偏移；每次洗白都应带来新误会或新任务。

### 《我在西幻世界独断万古》

卖点是前世健美/武道执念者转生西幻，以东方修炼认知重新解释魔法体系，把魔法回路当经脉，把冥想当筑基。

- 第1-3章：通过前世死亡与西幻成长环境建立主角的修炼执念，确认剑与魔法世界并展示身体基础。
- 第4-5章：主角接触魔法，开始用武道逻辑理解施法；贵族少女艾琳娜成为知识入口和关系锚点。
- 第6-7章：主角尝试“伪筑基”，冒险食用致幻蘑菇获得精神突破，在荒诞代价中学会火球术。
- 第8章：天赋检测显示全属性极高，主角提出创建势力、保护并培养艾琳娜，把个人修炼转向组织野心。
- 第9-10章：用时间跳跃完成多年沉淀、收徒、创立宗门和禁咒级战绩；十八岁以F级冒险者身份出山，与勇者线汇合。

结构要点：体系嫁接要有清晰映射和试错代价；时间跳跃必须结算关系、势力、战力和新身份，而不是一句“多年后”。

### 《早知道不这么玩了！》

卖点是玩家穿进多个游戏融合后的西幻学院世界，成为名声恶劣的反派未婚夫，并通过无限死亡式战斗空间快速补足实战。

- 第1章：先以龙族女皇的七百多年裂痕埋远古感情与历史线。
- 第2章：主角刚穿越就被公主提出决斗解除婚约，三天倒计时锁定开篇目标。
- 第3章：主角查阅历史，发现两个游戏世界被拼接为过去与未来，世界观形成第一层反转。
- 第4章：图书馆偶遇神秘学妹，以外界传闻和真实表现的反差重塑主角形象。
- 第5-7章：进入三尊雕像空间，被镜像反复杀死，以痛苦训练获得战斗本能；皇帝与大公开始关注反常变化。
- 第8-10章：主角把死亡经验转化为决斗实力，学院身份冲突、皇权观察和远古龙族线逐渐汇流。

结构要点：融合世界要先让读者认清“哪段是历史、哪段是当前、哪段将发生”；训练外挂必须有强烈痛感和明确收益。

### 《灰雾之上》

卖点是十余人集体穿越灰雾世界，靠玉册建立安全圈；主角以寿命向神秘羊皮纸购买序列配方和情报，踏上超凡求生。

- 第1-3章：集体穿越、诡异紫火与神秘大殿连续出现，群体在完全陌生环境中寻找共同规则。
- 第4章：玉册与众人产生联系，并将他们再次传送到灰雾荒野，形成最初安全屏障。
- 第5-6章：诡线鱼几乎团灭探索者；主角发现火焰克制并获得羊皮纸，以二十年寿命购买序列9“心光”配方。
- 第7-8章：众人开始建造营地，发现猫族文明；主角决定独自追逐超凡材料。
- 第9章：主角焚烧诡线鱼取材，却遭诅咒，真实寿命只剩三十七天，成长目标升级为硬倒计时。
- 第10章：主角离开安全圈寻找剩余材料，灰雾会扰乱方向，新的巨大威胁逼近。

结构要点：群体线负责文明建设和情感锚点，个人线负责冒险和超凡突破；寿命交易必须持续产生道德与生存压力。

### 《穿越异世界，我能看见经验条》

卖点是矿藏猎人穿进蒸汽与诡异并存的港口，能从学习、诊疗、调查和直面异常中获取经验，以医生身份追查海洋感染。

- 第1章：人脸鱼尸、召唤仪式、医生日记与“异梦”感染同时出现，主角继承一个危险诊所。
- 第2-4章：接触禁忌书卷，发现黑暗中存在怪物和“待在光里”的城市规则；经验条开始奖励理智与知识。
- 第5章：主角通过反复实验配制药物，把经验机制转化为谋生能力，并重新开张诊所。
- 第6-7章：从病人伤口、流浪汉和码头儿童处搭建情报网，发现铁鲸号、人形鱼和神秘收购者线索。
- 第8-9章：用报纸拼出教会进驻与感染扩散的时间线；房东劳森承认反复梦见大海，证实异梦已进入本地社会。
- 第10章：主角用稀缺光银草换取金钱与协助，并准备前往教会，调查线从私人诊所接入城市权力中心。

结构要点：调查流每章都要新增一条证据或排除一个错误判断；经验条既奖励战斗，也应奖励观察、分析、配药和社会经营。

### 《西幻：从鹰人部落到天使帝国》

卖点是玩家穿成太古时代弱小鹰人，先用现代生产知识发展部落，再以泰坦与龙之血觉醒超凡，目标从保族升级为建立文明。

- 第1-2章：鹰人部落被十年征召榨取，主角成为族长，并锁定传说中沐浴神血的超凡机缘。
- 第3章：七年时间推进木屋、水池、青铜和人口建设，为战争与种族升级做准备。
- 第4-5章：征召如期而至，主角带五十名族人提前进入瀚海森林，伪装王庭斥候寻找机缘。
- 第6章：主角找到老龟并沐浴泰坦与龙之血，血脉、属性、雷霆能力和天赋同时跃迁。
- 第7章：泰坦斩杀蓝龙王，这场胜利被主角认出为世界大战的导火索，个人机缘立刻接上纪元危机。
- 第8-9章：主角学习职业构建规则，见识王庭压迫与泰坦威严，产生取而代之的文明野心。
- 第10章：回归部落创立“御雷者”职业，玩家系统与NPC模板发生异常，完成从族长到职业开创者的第一跳。

结构要点：文明种田不能只有生产清单，建设必须服务战争、人口、职业和制度跃迁；每次时代跳跃都要结算损失与新目标。

### 《西幻：我有一本冒险者指南》

卖点是邪教贵族逃亡后沦为底层冒险者，指南书把任务转成经验；故事用写实资源计算和队友猜忌呈现残酷冒险生态。

- 第1章：从邪教献祭中逃生，失去贵族生活，靠最低级哥布林任务谋生；冒险者指南正式出现。
- 第2-3章：接下巨蜘蛛高额任务，加入临时队伍；通过迷雾小径规则展示冒险者的死亡率和互不信任。
- 第4-5章：小队搭建阵地、与巨蜘蛛正面作战，伤亡发生；死亡同伴的遗产立刻被瓜分，奠定底层世界基调。
- 第6-7章：深入蛛巢、搜刮战利品，法师揭示蜘蛛异常进化；收益分配引发队伍对立。
- 第8-10章：返程路上的伏击、野兽与内部冲突升级，主角依靠谨慎、指南收益和战斗判断争取活着回城。

结构要点：底层冒险流的爽点来自“比别人多准备一步、多看懂一条规矩”；金钱、负重、药剂、装备损耗和战利品分配都能制造剧情。

### 《领主：我在苦痛世界，养成少女》

卖点是被家族流放到诅咒领地的男爵，拥有识别潜力的小地图/养成能力，能把被轻视的亚人少女培养为神选者并开荒领地。

- 第1-2章：主角被迫前往死亡率极高的夜幕领，与家族和弟弟发生冲突，确立无兵、缺钱、被针对的开荒困境。
- 第3-5章：主角购买亚人奴隶阿诗娜，发现其潜力并改善待遇；通过衣食和魔药建立最初信任与养成关系。
- 第6-7章：路遇腐尸与难民，地图提前预警；护送骑士和边境贵族失职，主角被迫亲自组织奴隶反击。
- 第8章：要塞拒绝开门并把主角当缓冲，主角决定放弃旧规则，主动武装奴隶、清理腐化区域。
- 第9章：阿诗娜觉醒凛冬女神神选者，主角获得第一名真正超凡核心部下。
- 第10章：队伍进入废弃关隘清剿腐尸，与幸存士兵发生冲突，主角开始用血与铁建立领主秩序。

结构要点：养成必须与领地生产、军事、安全和忠诚绑定；改善待遇只是起点，前10章就要让被养成人物兑现战斗或治理价值。

## 横向类型矩阵

### 巫师/职业成长

- 重点：资质、导师、资源、知识成本、职业晋升。
- 开篇闭环：发现天赋 -> 验证能力 -> 获得进入巫师/职业体系的资格。
- 长线升级：学徒组织 -> 学院/高塔 -> 地区战争 -> 位面或神祇秩序。

### 魔法学院/恶役求生

- 重点：原作信息差、声誉修复、女主关系、学院考核、家族政治。
- 开篇闭环：必死身份 -> 剧情偏移 -> 第一次洗白/决斗 -> 更大角色注意。
- 长线升级：学院事件 -> 王国政治 -> 教会/魔族 -> 原作终局改写。

### 克苏鲁/诡异调查

- 重点：规则、证据链、理智、感染、教会或守夜组织。
- 开篇闭环：异常尸体 -> 城市规则 -> 第一名病人/怪物 -> 感染确认。
- 长线升级：街区病例 -> 港口源头 -> 教会隐瞒 -> 海外遗迹/古神复苏。

### 冒险者/地下城生存

- 重点：装备、补给、地图、队伍信任、战利品、雇主成本。
- 开篇闭环：接任务 -> 入危险区 -> 伤亡与收益 -> 带着新隐患返程。
- 长线升级：城镇委托 -> 大型地下城 -> 冒险团/公会 -> 王国级灾难。

### 种族文明演化

- 重点：人口、技术、职业、制度、战争、文明认同。
- 开篇闭环：弱族危机 -> 基础建设 -> 获得超凡种子 -> 创立首个职业。
- 长线升级：部落 -> 王国 -> 帝国 -> 种族神系或世界秩序。

### 领主开荒/养成

- 重点：安全、粮食、人口、核心部下、军事、领地合法性。
- 开篇闭环：被流放 -> 招募第一批人 -> 觉醒核心部下 -> 占领第一个据点。
- 长线升级：废土据点 -> 稳定领地 -> 贵族战争 -> 帝国/神权博弈。

## 后续选题建议

优先选择“西幻熟悉框架 + 一个反常识职业/规则 + 可扩张组织”。

- 巫师知识流：`巫师学院只收天才，我靠给魔法写注释晋升`。卖点是把复杂术式拆成可复制教材，最终改变知识垄断。
- 恶役学院流：`全校都等我按剧情黑化，我先承包了地下城后勤`。卖点是必死反派靠经营地下城反向控制学院资源。
- 克苏鲁医生流：`港口所有病人都梦见海，而我能给梦境做手术`。卖点是病例单元、梦境规则和海神感染主线。
- 序列调查流：`每破解一条禁忌，我就能删掉一种超凡代价`。卖点是调查证据直接改造职业体系。
- 冒险者经营流：`我不下地下城，只给冒险团做死亡复盘`。卖点是情报、装备方案和失败经验变现，逐步建立公会。
- 种族文明流：`我在巨龙时代把地精教成了工程师`。卖点是弱族工业化、职业创造和龙族战争。
- 领主养成流：`我的领民全是被神明判废的候选者`。卖点是识别隐藏天赋，用领地制度批量培养“废物”。
- 教会反套路流：`我被选为圣子后，发现神谕可以讨价还价`。卖点是神权规则、信仰结算和教会政治。
- 骑士血脉流：`别人服魔药觉醒，我给祖先的血脉补完族谱`。卖点是家族谱系、血脉考古和骑士战争。
- 世界融合流：`三个西幻游戏合服后，只有我知道版本冲突点`。卖点是历史、当前与未来三套规则互相打架。
- 黑暗喜剧流：`我在亡灵城开心理诊所，病人都怕我治好它们`。卖点是职业认知错位、怪物单元剧和城市声望。
- 帝国开拓流：`帝国把我流放到地图边缘，我在那里发现了第二轮太阳`。卖点是边疆领地、异族文明和宇宙级世界秘密。

## 大纲制作要求

### 必填信息

- 一句话卖点：用一句话同时说明西幻框架、主角差异和主要生存/成长矛盾。
- 主角初始困境：明确谁要杀他、什么在倒计时、为什么不能回头。
- 核心机制：触发条件、收益形式、限制、代价、升级方式和可重复剧情接口。
- 西幻体系：只先确定与前30章直接相关的职业、组织、种族、货币或爵位规则。
- 第一章钩子：开篇3段内出现异常物、死亡威胁、身份审判或无法拒绝的任务。
- 前10章跃迁：主角必须拿到第一项真正改变处境的能力、身份、伙伴或据点。
- 前30章目标：明确要进入哪个组织、解决哪个危机、占领哪个据点或晋升哪个职业。
- 第一反派/阻力：必须是可行动的具体对象或危机，不要只写“世界很危险”。
- 长线悬念：神祇、战争、感染、原作偏移、文明崛起或世界边界至少选择一条。

### 前30章模板

- 1章：异常开局，亮出生存威胁与主角差异。
- 2章：核心机制第一次运转，主角做出不可逆选择。
- 3章：获得小收益，同时暴露规则限制或代价。
- 4-5章：建立本地规则，接触第一位盟友、病人、导师、队友或部下。
- 6-8章：主动执行高风险验证，展示能力但引来更大注意。
- 9-10章：第一次跃迁，获得职业、身份、伙伴、据点或关键情报；抛出大钩子。
- 11-13章：进入学院、教会、公会、城镇、王庭、领地或更深地图。
- 14-16章：第二轮规则学习，出现内部利益冲突或组织门槛。
- 17-20章：主角用第一阶段积累解决组织内具体问题，得到部分认可与敌意。
- 21-24章：第一场大危机爆发，个人线与世界线正式汇合。
- 25-27章：主角主动布局，完成越级破局、调查揭露、职业晋升或据点夺取。
- 28-30章：结算资源、关系、身份与代价；打开下一卷地图和更高层敌人。

### 长线卷纲模板

- 第一卷：活下来并进入体系。完成核心机制、第一伙伴和第一组织入口。
- 第二卷：在组织中站稳。解决学院、公会、教会、领地或族群内部问题。
- 第三卷：组织外部战争。对抗相邻势力、地下城灾难、感染扩散或王国征召。
- 第四卷：世界历史翻面。揭示神祇、古代战争、游戏融合或超凡序列真相。
- 第五卷：主角建立自己的秩序。创立职业、势力、领地、种族制度或新教义。
- 后续卷：按地图、文明层级和敌人层级同步扩大，避免只做数值换皮。

### 检查清单

- 第1章前3段是否出现了异常或死亡压力？
- 第3章前核心机制是否已经实际运转？
- 第5章前是否获得一次可见收益？
- 第8章前是否进行一次高风险主动行动？
- 第10章是否完成能力、身份、伙伴或据点跃迁？
- 第20章前是否进入更大的社会组织或地图？
- 第30章是否既解决一个具体危机，又打开更大世界矛盾？
- 每3章是否至少新增一项信息、收益、关系、敌人或悬念？
- 西幻名词是否随事件出现，而不是集中百科式说明？
- 金手指是否有成本、限制和持续制造剧情的接口？
- 主角是否主动利用规则，而不是等待任务和救援？
- 若引用样本，是否明确只依据前10章，没有把推演写成原作事实？
