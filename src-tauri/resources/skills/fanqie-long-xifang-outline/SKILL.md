# 番茄长篇西方奇幻大纲

## 工作流程

1. 先确认目标产物：在 `选题方向`、`核心设定`、`样本节奏分析`、`前30章大纲`、`长线卷纲`、`后续剧情推进` 中选择一种；如果用户已经说清楚，就直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-long-xifang-patterns.md`。需要借鉴本地样本时，明确这些材料只收录每部作品前10章，不把推测的后续剧情写成已验证事实。
3. 每个想法都要锚定一个清晰卖点：高压生存身份、可重复成长机制、西幻职业/种族/组织体系，以及能持续扩张的地图或文明冲突。
4. 围绕“危机—验证—跃迁—开图”搭大纲：第1章亮危机与核心差异，第2-5章验证规则，第6-10章完成第一次能力或身份跃迁，第11-20章接入组织/城镇/学院/族群，第21-30章收束第一场大冲突并打开更大的世界矛盾。
5. 输出可直接使用的写作产物，不写空泛理论；优先给出逐章事件、冲突来源、成长反馈、世界信息和章末悬念。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 每3-5章都要让主角目标可见，并至少兑现一次信息、资源、战力、身份或关系收益。
- 制作大纲时使用具体章节标题。
- 每章都包含 `主要事件`、`成长/爽点`、`章末悬念`。
- 西方奇幻名词要服务剧情，不要连续堆砌王国、教会、神祇、种族、爵位和魔法术语。
- 前10章必须至少完成一次规则验证和一次阶段跃迁，不能只介绍世界观。
- 不要照搬样本剧情、人名、章节标题、专有名词或标志性措辞，只提取结构和机制。
- 分析样本时把内容分为 `样本已展示` 与 `后续可推演`；后者必须明确标注为创作推演。
