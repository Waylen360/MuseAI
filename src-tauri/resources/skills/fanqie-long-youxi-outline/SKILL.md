# 番茄游戏体育大纲

## 工作流程

1. 先确认目标产物：在 `选题方向`、`核心设定`、`样本节奏分析`、`前30章大纲`、`长线卷纲`、`后续剧情推进` 中选择一种；用户已经说清楚时直接执行。
2. 生成选题、节奏分析或大纲前，先阅读 `references/fanqie-long-youxi-patterns.md`，再判断属于 `全民求生`、`传统网游/游戏异界`、`游戏降临` 或 `体育竞技` 哪条路线。
3. 每个想法都要锚定一个可反复兑现的核心循环：探索与结算、刷怪与成长、交易与组织竞争、训练与比赛验证，不能只靠一次性外挂撑开篇。
4. 围绕前30章短闭环搭大纲：第1章同时给危机和卖点，第2-5章验证核心机制，第6-10章完成第一次能力/身份跃迁，第11-20章引入榜单、组织、站点、俱乐部或区域竞争，第21-30章完成第一场大事件并打开下一阶段。
5. 输出可直接使用的写作产物；优先给出具体章节标题、主要事件、规则变化、收益结算、外部反应和章末悬念，不写空泛理论。

## 输出规则

- 除非用户另有要求，否则全部使用中文。
- 明确区分 `样本开头已出现的内容` 与 `根据类型规律推演的后续方向`；本地范文只有开头片段，不把推演写成原作事实。
- 每3-5章完成一次可见的小目标；每8-10章至少有一次能力、载具、职业、身份、排名或球队地位的升级。
- 制作大纲时使用具体章节标题；每章包含 `主要事件`、`兑现/爽点`、`章末钩子`，涉及机制时补充 `规则或资源变化`。
- 游戏线必须写清数值、资源、风险、交易或组织反应；体育线必须写清训练指标、比赛任务、技术验证、队友/教练/媒体反应。
- 不照搬样本人名、怪物、载具、技能、俱乐部桥段、章节标题或标志性措辞，只提取节奏、结构和机制。
